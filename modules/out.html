Audio Output (Out) Module

Output

The output module is a critical module for a virtual modular synthesiser, as it is the endpoint, or final modular, for any synth configuration/module chain. The output module takes the processed audio signal from the other modules and outputs the actual sound through the speakers of the device the user is running the virtual modular synthesiser from.

There is a maximum of 1 audio output node per project.

This module takes 1 inputs and produces 0 output:
Input 1 is the audio signal that you wish to output to your speakers.
